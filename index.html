🥜 Peanut butter scaling guide

Mon — Wildcard breakfastNo PB target. Poha, upma, cheela, thepla — whatever the kitchen offers.
Tue / Thu — Run 10–12 kmPB 20g + banana. ~390 kcal breakfast. Banana pre-run or post-run.
Wed — Easy / short hikePB 15g base. Lighter day, standard bowl.
Fri — Cycling 30–50 kmPB 25g + banana. ~460 kcal. On-bike: dates + peanut chikki.
Sat — Hike 10–15 km, 800–1400mPB 25g + banana. Trail snacks: peanuts, chikki, dates — carry more than you think.
Sun — Long run 21–26 kmPB 30g + banana. ~510 kcal. Maximum version. Choc milk recovery within 30 min.

🍷 Alcohol reintroduction guide

Best slotsFriday evening (post-cycling, next hard session is hike not run). Sunday evening (long run done, next run is Tuesday).
ProtectSaturday night — pre-long-run sleep and glycogen storage matter. Keep Saturday alcohol-free.
Quantity1–2 drinks, 2–3 evenings per week. Fits within snack budget (~150–200 kcal per drink).
Phase inWeeks 1–2: Fri + Sun only. Weeks 3+: add Wed evening if desired (easy day, no impact on Thu run).
Built for a 47yr vegetarian marathon runner · Hal Higdon Senior plan · targeting 63 kg
Gradual step-up from 1,800 kcal baseline over 6 weeks · Small deficit preserved on Mondays
The belly responds to lower cortisol from better fuelling, not more restriction.
